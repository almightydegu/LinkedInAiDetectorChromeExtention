LinkedIn AI Detector Chrome Extension Settings

Claude API Key
Your key is saved locally using Chrome's encrypted sync storage and is only sent directly to Anthropic's API — it is never transmitted to any other server.
How to get a Claude API key
  1. Go to console.anthropic.com and sign in (or create a free account).
  2. In the left sidebar, click API Keys.
  3. Click Create Key, give it a name (e.g. "LinkedIn Detector"), and copy it.
  4. Paste it into the field above and click Save Key.

Cost: This extension uses Claude Haiku 4.5 — the fastest, most affordable model. Each analysis costs roughly $0.001 (one tenth of a cent), based on ~600 input tokens and ~40 output tokens per request at current API pricing. Analysing 100 posts costs about 10 cents; 1,000 posts about $1.

New accounts receive free credits that are more than enough to try this extension.

Display Settings
Icon colour mode
Controls how the robot and human icons are coloured.
Show percentage
Displays the numeric score next to the label. Hovering over the badge always shows the percentage regardless of this setting.

Posts analysis mode
Controls how posts are analysed for AI content.
Comments analysis mode
Controls how comments are analysed. Each comment uses one API call, the same cost as a post.
Minimum text length (Auto mode only)
In Auto mode, posts or comments shorter than these thresholds are silently skipped — no badge, no API call. In On-demand mode the "🔍 Check for AI" pill is always shown regardless of length.
Understanding the score

Each post gets a score from 0–100% indicating how likely it was written by AI. Five icons are shown next to the post author — robots for AI, humans for human writing.

0–20% — Human written (5 humans)
21–40% — Likely human (1 robot)
41–60% — Uncertain (2 robots)
61–80% — Likely AI (3 robots)
81–90% — Mostly AI (4 robots)
91–100% — AI written (5 robots)
